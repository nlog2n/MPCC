                      Mpcc For Win95 驱动程序安装说明1 打开左下角”开始“菜单，选择”设置|控制面板“，双击”网络“图标。2 选中”配置“项，选择”添加“按钮。双击”适配器“。选择”从磁盘安装“按钮。  确定为源盘A:\。在此过程中，会要求输入win95 cd-rom 路径，以拷入相应的系统    协议文件。3 如果还未安装Tcp/ip，回到网络配置，安装协议，选择Microsoft下的Tcp/ip项。结  束后修改Tcp/ip属性，设置ip地址及掩码等参数。4 修改mpcc属性。双击”Mpcc netcard"项，选择“资源”，按网卡硬跳线填写参数，  确保硬件资源不发生冲突：                       I/O 地址        ：300h (缺省)                        IRQ 号          ：5    (缺省)            RamBase地址     ：D0000(缺省) 5 选择“高级”项，填写与 x.25相关的参数配置。  一般只需修改如下参数：(举例)                    波特率        BaudRate         : 9600 bps   时钟          LINKCLK0         : CLK_EXT(外时钟)   设备类型      LINKDEV0         : DEV_DTE   内存映象地址  RamBaseAddress   : D0000   X121地址      X121ADDRESS      : 11110015    x121地址与ip地址对应表AddrPair :+128.1.1.5 11110015    其中格式按照：每项以+开头，后接ip地址，空格，最后是x121地址，如表中     有三项写作 +128.1.1.3 11110013 +128.1.1.4 11110014 +128.1.1.5 11110015 6 选择“确定”，重启动计算机，网卡生效。 附：一  Mpcc 卡上硬跳线  1.I/O  地址表    RamBaseAddress   : 内存映象地址         X121ADDRESS      : X121地址          AddrPair         : X121地址与ip地址对应表,用于路由。         FRAMELEN0        : 最大帧长度          LINKADDR0        : 逻辑通道号(LCN),一般从1到255。         LINKMODUL0       : 链接取模，一般为8或128。         LINKTYPE0        : 链接类型           LINKWIN0         : 窗口大小，为最大未确认帧数。         LINKN20          : 重发次数          LINKT10          : T1计时器的超时时间。                                                 方 晖                                                 1997.11.24
